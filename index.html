Learning Submodular Objectives for Improved Image Retrieval
Computer Vision Lab, ETH Zurich

We formulate image retrieval as a subset selection problem and we address it using submodularity. In order to select the subset that is maximally relevant and representative, we learn an objective function as a linear combination of submodular functions.

    Visual Interestingness of Images
    Computer Vision Lab, ETH Zurich

    We analyze how image content and emotions are linked to interest in order to get new insights on what makes an image interesting. We build a predictive model using deep convolutional networks, which predicts interest more accurately that the previous state-of-the-art.

    Assessing state of alertness using wearable systems
    Indian Institute of Technology, Kharagpur

    Reduced alertness of a person engaged in operations such as flying an aircraft, driving or in cases of critical surgical operations may result in harmful consequences. Considering this premise, we develop a smart-phone based portable solution to monitor/ classify the state of alertness of an individual. We extract several physiological and psychological paramters and train an SVM on these features that can accurately predict the state of alertness.

    Salient Object Detection via Objectness Measure
    Video Analytics Lab, IISc Bangalore

    Salient object detection has become an important task in many image processing applications. The existing approaches exploit background prior and contrast prior to attain state of the art results. In this paper, instead of using background cues alone, we estimate the foreground regions in an image using objectness proposals and utilize it to obtain smooth and accurate saliency maps.

    Automated Essay Scoring
    IIT Kharagpur

    We propose a regression based approach for automatically scoring essays written in English. We use standard NLP techniques for obtaining the features from the text and integrate it with an existing framework that uses vector space model to improve the results. We show that our approach yields results that are comparable to professional human raters while at a much faster rate.

    Selective Search for Object Recognition (Python Implementation)

    Selective Search is an object proposal algorithm which combines the strength of both exhaustive search and segmentation. We provide a python implementation of selective search. We also integrate it with fast-rcnn, which uses the proposals generated by selective search for object recognition.

    Comparative Analysis of Signal Processing Algorithms for Bearing Fault Diagnosis

    This project aims at comparing how effective different Signal Processing algorithms are, in detecting bearing faults from noisy signals. Many algorithms such as envelope detection, Empirical Mode Decomposition, FFT and techniques using morphological operators etc were implemented and their performances were evaluated on data obtained from CWRU bearing data center.
